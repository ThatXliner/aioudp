> A better API for asynchronous UDP

> Note: This library only supports asyncio. If you need to use other async frameworks, I highly recommend you use [AnyIO]() instead

## Installation

You can get this project via `pip`

```bash
$ pip install aioudp
```

Or, if you're using [Poetry](https://python-poetry.org)

```bash
$ poetry add aioudp
```
